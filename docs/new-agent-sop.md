# New Agent SOP

For a new operator or agent integrating OpenClawBrain:

1. Read [learning-first-convergence.md](learning-first-convergence.md)
2. Follow [openclaw-attach-quickstart.md](openclaw-attach-quickstart.md)
3. Read [openclaw-integration.md](openclaw-integration.md)
4. Verify [operator-observability.md](operator-observability.md)
5. Reproduce current proof boundaries with [reproduce-eval.md](reproduce-eval.md)

Keep the learning-first invariant in mind: serve from the promoted pack, and when learned routing is required, prove the active `route_fn` actually ran.
